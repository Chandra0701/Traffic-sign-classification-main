🚦 Traffic Sign Recognition using Deep Learning

A deep learning-based traffic sign recognition system built using TensorFlow/Keras that accurately classifies traffic signs from image datasets and supports real-time prediction.

🔍 Overview

This project focuses on building a robust Convolutional Neural Network (CNN) model to classify traffic signs across multiple categories. It includes a complete pipeline from data preprocessing to model training and evaluation, ensuring high accuracy and scalability.

⚙️ Tech Stack

Python

TensorFlow / Keras

OpenCV

NumPy

scikit-learn

🚀 Features

📌 End-to-end deep learning pipeline for traffic sign classification

📌 Image preprocessing (resizing, normalization, augmentation-ready)

📌 One-hot encoding and stratified train-test split for balanced training

📌 Optimized CNN architecture for high accuracy

📌 Modular and maintainable code structure

📌 Logging and error handling for debugging and reliability

📌 Real-time prediction capability (via OpenCV)

🧠 Model Details

Built a CNN architecture tailored for multi-class image classification

Tuned hyperparameters (learning rate, batch size, layers) for performance

Achieved high classification accuracy across diverse traffic sign categories
